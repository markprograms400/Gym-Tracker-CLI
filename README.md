# Gym-Tracker-CLI
A command-line Python application for tracking gym workouts. Users can log exercises with weight, sets/reps, and date, and view their workout history. Data is stored locally using a JSON file, allowing progress to persist across sessions.

**Features**
- Record workouts with exercise name, weight, sets/reps, and date
- View workout history grouped by exercise
- Persistent local storage using data.json
- Menu-driven command-line interface

**Technologies**
- Python
- JSON for data storage

**Overview**
This project was developed to strengthen core programming concepts, including file handling, data structures, and user input management. It demonstrates the ability to build a structured, interactive CLI application with persistent data.

**Future Improvements**
- Improved output formatting for readability
- Input validation and error handling
- Personal record (PR) tracking
- Filtering and searching workouts
